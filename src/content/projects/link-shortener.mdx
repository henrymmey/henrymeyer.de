## Overview

Link Shortener is a private URL shortener built for speed, control, and minimal operational overhead.
It runs on Cloudflare Workers and is designed around simple, durable workflows.

## Core Features

- Fast redirect handling on edge infrastructure
- Controlled mapping management for short links
- Lightweight JavaScript implementation with low complexity

## Operating Principles

The project is intentionally pragmatic: clear architecture, quick deploys, and transparent behavior.
It is built for personal and team usage where reliability matters more than feature bloat.

## Current Focus

Current improvements focus on administration ergonomics, validation rules, and better visibility into redirect usage patterns.
